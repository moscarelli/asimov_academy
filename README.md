# Asimov Academy

This repository is a starting point for the Asimov Academy project.

## What's included

- A Python-focused .gitignore to keep the repo clean.

## How to use

- Create a virtual environment and install dependencies (if any):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

- Run tests (if present):

```powershell
pytest
```
