# Correcting Plan Errors

## Overview

The Employee Plans Compliance Resolution System (EPCRS) offers three programs for correcting plan errors:

*   Self-Correction Program (SCP)
*   Voluntary Correction Program (VCP)
*   Audit Closing Agreement Program (Audit CAP)

### Find Plan Errors

You can fix common mistakes and file a VCP application.

### Fix Plan Errors

Fix common mistakes, including:

*   Failing to adopt a new pre-approved defined contribution plan by the April 30, 2016 deadline
*   Missing required minimum distribution failures
*   Not filing Form 14568

## Fix-it Guides

Submission kits are available for VCP fill-in forms. Use the following guides:

*   Revenue Procedure 2021-30 (effective July 16, 2021)
*   Revenue Procedure 2019-19 (effective April 19, 2019)
*   Revenue Procedure 2008-50 (superseded as of April 1, 2013)

## Avoid Plan Errors

To avoid common retirement plan errors:

*   Follow revenue procedures
*   Check the IRS website for updates and guidance
