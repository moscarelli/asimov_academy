# Fix-it Guides - Common Problems, Real Solutions

Common problems in retirement plans can be fixed with the right guidance. Each guide provides:

*   An overview of the rules for each plan type
*   An overview of the Employee Plans Compliance Resolution System
*   The most frequent errors we find in each plan type
*   Tips on how to find, fix and avoid these mistakes

## 401(k) Plan Fix-it Guide

The guide provides an overview of the 401(k) plan rules, including the requirements for employer matching contributions and vesting schedules.

### Common Issues:

*   **Inadequate Vesting Schedules**: The guide helps plan sponsors identify and correct vesting schedule issues that can result in unexpected distributions.
*   **Insufficient Fiduciary Oversight**: The guide provides tips on how to ensure fiduciary compliance, including the importance of regular plan audits.
