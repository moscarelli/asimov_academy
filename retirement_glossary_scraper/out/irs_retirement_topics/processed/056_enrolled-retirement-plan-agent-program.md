# Enrolled Retirement Plan Agent Program

## Overview

The Enrolled Retirement Plan Agent (ERPA) program is designed to help plan sponsors and their agents navigate the complexities of ERPs. The program provides a framework for ERPs, including rules, regulations, and procedures.

### Filing Requirements

*   File Form 5500-SR with the IRS within 7 months after the end of each tax year.
*   File Form 5500-EZ with the IRS if there is no need to file a full Form 5500-SR.

### Reporting Requirements

*   Report any changes in the plan's assets, liabilities, or income on Form 5500-SR within 2 months after the change occurs.

### Agent Responsibilities

*   Agents must complete annual training and certification requirements.
*   Agents must maintain accurate records of plan information and submit them to the IRS on a regular basis.

## Forms and Instructions

*   **Form 5500**: Annual Report
*   **Form 5500-SR**: Simplified Annual Return of Employee Benefit Plan
*   **Form 5500-EZ**: Annual Return of Qualified Domestic Retirement Plans (QDRPs)
*   **Form 5500 Instructions**: Instructions for Form 5500
*   **Form W-9**: Request for Taxpayer Identification Number and Certification

## Enrolled Retirement Plan Agent Program Changes

The ERPA program is subject to change. Agents must stay up-to-date on any changes to the program.

### Annual Filing Season Program Participants

Agents who participate in the Annual Filing Season Program (AFSP) are eligible for a reduced filing fee.

*   **Form 5500-SR**: Simplified Annual Return of Employee Benefit Plan
*   **Form 5500-EZ**: Annual Return of Qualified Domestic Retirement Plans (QDRPs)

### Certified Professional Employer Organization (CPEO)

CPEOs are entities that provide administrative services to small businesses. CPEOs must be certified by the IRS.

*   **Form 5500-SR**: Simplified Annual Return of Employee Benefit Plan
*   **Form 5500-EZ**: Annual Return of Qualified Domestic Retirement Plans (QDRPs)

### Modernized e-File

The modernized e-file system allows for faster and more efficient processing of ERP filings.

*   **Form 5500-SR**: Simplified Annual Return of Employee Benefit Plan
*   **Form 5500-EZ**: Annual Return of Qualified Domestic Retirement Plans (QDRPs)

### E-File Providers

E-file providers are entities that provide e-filing services for ERPs. E-file providers must be certified by the IRS.

*   **Form 5500-SR**: Simplified Annual Return of Employee Benefit Plan
*   **Form 5500-EZ**: Annual Return of Qualified Domestic Retirement Plans (QDRPs)

### Enrolled Actuaries

Enrolled actuaries are professionals who have been certified by the IRS to provide actuarial services.

*   **Form 5500-SR**: Simplified Annual Return of Employee Benefit Plan
*   **Form 5500-EZ**: Annual Return of Qualified Domestic Retirement Plans (QDRPs)

### EITC Central

EITC central is a resource for information on the Earned Income Tax Credit (EITC).

*   **Form 1040**: Individual Income Tax Return
*   **Form W-9**: Request for Taxpayer Identification Number and Certification
