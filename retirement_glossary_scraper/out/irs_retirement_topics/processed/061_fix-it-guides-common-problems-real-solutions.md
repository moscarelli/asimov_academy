# Fix-it Guides - Common Problems, Real Solutions

## Overview

This guide provides tips on how to find, fix, and avoid common mistakes in retirement plans. Each guide covers a specific plan type and includes an overview of the rules, errors found, and tips for users.

## 401(k) Plan Fix-it Guide

### What You Need to Know
- An overview of the 401(k) plan rules
- Common errors to watch out for
- Tips for avoiding mistakes

### How to File

*   When to file: Typically, individuals can start filing their tax returns by January 31st.
*   Where to file: Individuals should file Form 1040 with the IRS.
*   Update your information:

    *   If you've recently changed addresses or phone numbers, update them on the IRS website.
