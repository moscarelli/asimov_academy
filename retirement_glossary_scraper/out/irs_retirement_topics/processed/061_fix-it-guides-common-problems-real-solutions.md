### 401(k) Plan Fix-it guide

Each guide provides:
* an overview of the rules for each plan type
* an overview of the Employee Plans Compliance Resolution System
* the most frequent errors we find in each plan type and
tips on how to find, fix and avoid these mistakes.

We have discontinued the PDF versions of each fix-it guide and would encourage you to only use the updated web-based version of each guide.

**Common Problems and Real Solutions**

* [Error 1: Incorrect employer ID number](#error-1)
* [Error 2: Missing or incomplete documentation](#error-2)
* [Error 3: Failure to report income](#error-3)
